# 🧮 Calculator Pro

A simple calculator application built using **Python** and **CustomTkinter**.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ⌨️ Keyboard support
- 🌙 Dark theme

## Technologies Used

- Python
- CustomTkinter
- PyInstaller

## Run the Project

Install CustomTkinter:

```bash
pip install customtkinter
```

Run the application:

```bash
python calculator.py
```
